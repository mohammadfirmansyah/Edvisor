Pengguna Mulai (Terminator)

- Login atau Lupa Kata Sandi atau Mendaftar Akun (Decision)
    - Login (Process)
        - Pengguna diarahkan ke langkah berikutnya setelah sistem berhasil melakukan validasi input pengguna.
    - Lupa Kata Sandi (Process)
        - Pengguna diarahkan ke halaman Login setelah sistem berhasil mengirimkan password baru ke email pengguna.
    - Mendaftar Akun (Process)
        - Pengguna diarahkan kembali ke halaman Login setelah sistem berhasil melakukan validasi input pengguna.

- Beranda (Process)

- Pengguna Memilih: Guru Model atau Observer atau Navigasi Utilitas (Decision)
    - Alur Guru Model (Subprocess)
        - Memilih Kelas Guru Model atau Membuat Kelas (Decision)
            - Memilih Kelas Guru Model (Process)
                - Melihat Hasil Observasi (Process)
                - Mengunduh Formulir (Process)
                - Selesai (Terminator)
            - Membuat Kelas (Subprocess)
                - Detail Kelas (Process)
                - Unggah Berkas (Process)
                - Memilih Observer (Process)
                - Memilih Nomor Siswa (Process)
                - Menyimpan Formulir (Process)
                - Menyalin Kode Kelas (Process)
                - Kembali ke Guru Model (Process)
                    
    - Alur Observer (Subprocess)
        - Memilih Kelas Observer atau Gabung Kelas (Decision)
            - Memilih Kelas Observer (Process)
                - Mengisi Formulir (Process)
                - Menyimpan Formulir (Process)
                - Selesai (Terminator)
            - Gabung Kelas (Subprocess)
                - Masukkan Kode Kelas (Input/Output)
                - Memilih Nomor Siswa (Process)
                - Menyimpan Formulir (Process)
                - Kembali ke Observer (Process)
        
    - Alur Navigasi Utilitas (Subprocess)
        - Bantuan atau Profil (Decision)
            - Bantuan (Process)
                - Selesai (Terminator)
            - Profil (Subprocess)
                - Menampilkan Profil (Process)
                    - Memperbarui Kata Sandi (Process)
                        - Selesai (Terminator)
                    - Memperbarui Data Profil (Process)
                        - Selesai (Terminator)
                    - Memperbarui Foto Profil (Process)
                        - Selesai (Terminator)
